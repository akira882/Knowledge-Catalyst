# Knowledge Catalyst (ナレッジ・カタリスト)

## 概要
本リポジトリは、**{Client}様**の持続的成長と生成AI活用能力の最大化を目指した、「生成AI学習コンテンツ最適化プラットフォーム」の導入提案資料群です。

現状の「膨大な学習コンテンツが埋没し、活用が進まない」という課題を解決し、**「静的な図書館」から「動的なメンター」への転換**を実現するための戦略的アプローチをまとめています。

## ディレクトリ構成とドキュメントリンク

```text
.
├── README.md               <-- 本ファイル
└── docs/
    ├── 01_PROPOSAL.md      -- 提案書本編：課題認識から解決策、期待効果
    ├── 02_ARCHITECTURE.md  -- 技術仕様案：RAG構成とデータパイプライン
    ├── 03_UX_DESIGN.md     -- 体験設計：ユーザー層別のインターフェース定義
    ├── 04_ROADMAP.md       -- 実装計画：導入ステップと継続的改善
    └── 05_DREAM_TEAM_REVIEW.md -- 第三者レビュー：技術専門家による評価結果
```

## 閲覧ガイド
1. [提案書本編 (01_PROPOSAL.md)](docs/01_PROPOSAL.md) を最初にご覧ください。本プロジェクトの核心となる価値提案を記載しています。
2. その後、具体的な実現手法として [技術仕様案](docs/02_ARCHITECTURE.md) と [UX設計](docs/03_UX_DESIGN.md) をご確認ください。
3. [ドリーム・チームによるレビュー (05_DREAM_TEAM_REVIEW.md)](docs/05_DREAM_TEAM_REVIEW.md) では、本案の客観的な実現可能性をご確認いただけます。
4. 最後に [実装計画 (04_ROADMAP.md)](docs/04_ROADMAP.md) にて、直近のステップをご確認いただけます。
